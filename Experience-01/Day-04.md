# Day 4 – Windows Update and Patch Management Project

Today is my fourth day, and I am starting a new project focused on Windows Update and patch management using Microsoft Intune.

## Project Overview

This project focuses on reviewing how Windows updates are currently managed across the organization's devices and identifying opportunities to improve the update deployment process.

The project will include reviewing existing Windows Update policies, analyzing device update status, identifying outdated or problematic devices, and designing a structured update deployment strategy using Microsoft Intune.

## Goal

The main goal of this project is to better understand Windows Update management in an enterprise environment and gain hands-on experience with Intune Update Rings, policy assignments, deployment groups, update monitoring, and troubleshooting.

The project may also include creating or improving update policies and testing changes through a controlled pilot deployment before applying them to a larger group of devices.
